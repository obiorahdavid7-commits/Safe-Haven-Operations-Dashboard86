# Safe Haven Business Operations Dashboard

An interactive Power BI dashboard built to monitor key business operations metrics, revenue performance, and team productivity.

## Dashboard Features
- **KPIs**: Total Revenue, Target Achievement %, Productivity %, Total Orders
- **Trend Analysis**: Daily Revenue Trend line chart
- **Team Performance**: Performance breakdown by team member
- **Exception Reporting**: Filtered table showing Open Issues by Department
- **Interactive Filters**: Date and Department slicers for dynamic filtering

## Tools Used
- Power BI Desktop
- DAX Measures for KPI calculations

## Key Metrics
- $1.9M Total Revenue
- 172.5% Target Achievement
- 370 Total Orders
- 103.6% Productivity
